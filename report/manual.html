📊 성과 리포트 사용법
시즌 선택 및 분석
1
분석할 Arena 시즌을 체크박스로 선택 (복수 선택 가능)
2
분석할 GAR 시즌을 체크박스로 선택 (복수 선택 가능)
3
선택한 시즌으로 분석 → 버튼 클릭
4
위반자 목록과 벌점/등급이 자동 표시됩니다
⚖️ 가중치 설정
분석 결과 위 벌점 가중치 설정에서 항목별 점수를 조정할 수 있습니다.
－ ＋ 버튼으로 조정하면 벌점과 등급이 실시간으로 반영됩니다.

기본값 — Arena 미달 1점 · 공동미션 위반 1점 · GAR 미달 1점
정렬 기능
벌점순 — 벌점 높은 순 (기본값)
위반횟수순 — 전체 위반 횟수 기준
Arena 미달순 — Arena 미달 횟수 기준
공동미션순 — 공동미션 위반 횟수 기준
GAR 미달순 — GAR 미달 일수 기준
툴팁 — 시즌별 상세 정보
Arena / GAR / 공동미션 배지에 마우스오버(PC) 또는 터치(모바일)하면
시즌별 점수와 달성 여부를 확인할 수 있습니다.
